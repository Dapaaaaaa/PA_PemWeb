# QRIS Payment Setup Instructions

## Upload QRIS Image

Untuk mengaktifkan pembayaran QRIS, upload gambar QRIS kamu ke:

**Path:** `assets/img/qris.png`

### Rekomendasi Gambar:

- Format: PNG atau JPG
- Ukuran: 400x400px hingga 800x800px
- Pastikan QR code terlihat jelas dan dapat di-scan

### Cara Upload:

1. Dapatkan QRIS dari bank/e-wallet kamu
2. Screenshot atau download gambar QRIS
3. Rename file menjadi `qris.png`
4. Upload ke folder `assets/img/`
5. Refresh halaman payment untuk melihat perubahan

### Test Placeholder:

Sementara ada placeholder image yang sudah dibuat di `assets/img/qris.png`
Ganti dengan QRIS asli saat deployment.
